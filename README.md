# breathe478

A terminal-based guide for the [4-7-8 breathing technique](https://www.drweil.com/health-wellness/body-mind-spirit/stress-anxiety/breathing-exercises-4-7-8-breath/) developed by Dr. Andrew Weil.

**Inhale** for 4 seconds · **Hold** for 7 seconds · **Exhale** for 8 seconds

## Install

```
cargo install --git https://github.com/jhheider/breathe478
```

Or download a binary from [Releases](https://github.com/jhheider/breathe478/releases).

## Usage

```
breathe478
```

Press `q` or `Esc` to quit.
