# breathe478

A terminal-based guide for the [4-7-8 breathing technique](https://www.drweil.com/health-wellness/body-mind-spirit/stress-anxiety/breathing-exercises-4-7-8-breath/) developed by Dr. Andrew Weil.

**Inhale** for 4 seconds · **Hold** for 7 seconds · **Exhale** for 8 seconds

> **Status: complete & shelved.** This does one thing — guide the 4-7-8 breath —
> and does it. It is intentionally limited to that single pattern and is no
> longer under active development. Generalizing it into a multi-pattern breathing
> coach (box, coherent, Wim Hof, physiological sigh) was considered and dropped:
> [`breathe-cli`](https://github.com/AtomicIntuition/breathe-cli) already covers
> that ground. Use this if you want a minimal, single-purpose 4-7-8 timer.

## Install

```
cargo install --git https://github.com/jhheider/breathe478
```

Or download a binary from [Releases](https://github.com/jhheider/breathe478/releases).

## Usage

```
breathe478
```

Press `q` or `Esc` to quit.
